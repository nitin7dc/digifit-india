### ğŸ§® BMR Calculator

A comprehensive Basal Metabolic Rate calculator that helps you understand your daily calorie needs at rest.

**Features:**

- **Accurate Calculations**: Uses the Mifflin-St Jeor Equation for precise BMR calculation

- **User-Friendly Interface**: Simple form with age, gender, height, and weight inputs

- **Detailed Results**: Shows your BMR with comprehensive explanations

- **Educational Content**: Learn about BMR, factors affecting metabolism, and how to use your results

- **Form Validation**: Ensures accurate input with real-time validation

- **Responsive Design**: Works seamlessly on all devices

**How to Use:**

1. Navigate to the BMR Calculator from the header menu

2. Enter your age, select gender, and input height (cm) and weight (kg)

3. Click "Calculate BMR" to get your results

4. Review the detailed explanation and recommendations

### ğŸ§® Advanced Calculator (Dialog)

A popup calculator that provides both BMR and TDEE (Total Daily Energy Expenditure) calculations.

**Features:**

- **Dual Calculations**: Calculates both BMR and TDEE

- **Activity Level Integration**: Considers your activity level for TDEE calculation

- **Popup Interface**: Quick access without leaving the current page

- **Comprehensive Results**: Shows calories needed for different activity levels

**How to Use:**

1. Click "Calculator" in the header menu

2. Fill in your personal details (sex, age, height, weight)

3. Select your activity level from the dropdown

4. Click "Calculate" to see both BMR and TDEE results

## ğŸš€ Getting Started

### Prerequisites

- Node.js (version 16 or higher)

- npm or yarn package manager

- Angular CLI

### Installation

1. **Clone the repository**

   ```bash

   git clone <repository-url>

   cd digifit-india

   ```

2. **Install dependencies**

   ```bash

   npm install

   ```

3. **Start the development server**

   ```bash

   npm start

   ```

4. **Open your browser**

   Navigate to `http://localhost:4200`

### Build for Production

```bash

npm run build

```

The build artifacts will be stored in the `dist/` directory.
